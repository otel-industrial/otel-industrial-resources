## OpenTelemetry Industrial Collector

This repository is intended to explore what an OpenTelemetry Industrial Collector might look like and what capabilities may be needed to support industrial, OT, IIoT, and legacy environments.

## Contributing

Ideas, use cases, requirements, and feedback are welcome.

* Industrial use cases and requirements
* Deployment patterns and reference architectures
* Ideas for Collector distributions and packaging
* Industrial protocol integrations and receiver bundles
* Security, networking, and operational considerations
* Feedback from manufacturers, system integrators, vendors, and end users

If you have an idea, requirement, or use case that could help shape the future direction of this project, please open an issue to start a discussion.

As the project evolves, contributions in the form of documentation, reference configurations, and code will be welcome.
